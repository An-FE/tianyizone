绘制表格获取HTML代码

  • 利用KindEditor编辑器，去除不需要的标签，留下表格的功能和查看HTML源码的功能，直接绘制表格而获得HTML源码。
  • 点击表格的图标后，在弹出层中输入相对应的值，即可马上看到表格内容，在单元格中右击还可以插入单元格以及删除行、列等操作。
  • 在可视化窗口中编辑完表格之后，再通过点击获取HTML代码。最终的HTML代码或许会有部分可以删除的属性，或者应该是th标签而显示td标签时，需要各位手动去修改调整。
  • 在HTML代码中会出现<br />标签，这个也需要大家手动删除。
  • 利用KindEditor编辑器实现这个小工具的主要是便于使用某些编辑器时没有像DreamWeaver那样绘制表格获得代码，然后直接用写HTML代码绘制表格又比较头痛。
  • 这个小工具仅仅只是为了能方便去绘制表格，最终还是不能像DreamWeaver等软件一样操作方便，小志我基本上就没装DreamWeaver这样的大软件了，就用这个给自己偷懒。

该编辑器的版本是KindEditor ©，小志我只是利用了其中的表格功能，并未做其他修改，有兴趣的朋友可以去查看更多的编辑器demo效果。
